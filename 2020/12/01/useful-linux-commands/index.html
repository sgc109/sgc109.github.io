useful-linux-commands

본 포스트에서는 유용한 리눅스 쉘 명령어 중에서도 서버를 운영할 때 특히 유용하게 사용할 수 있는 명령어들을 알아볼 것이다.

새로운 팀에 합류하거나 새로운 서버를 사용하게되는 경우, 서버가 현재 어떤 상황에 처해있는지 빠르게 파악하고 싶을 때가 많다.

  • 전체 메모리 사용량 확인

  • top 명령 활용하기

    • E - 사이즈 단위(KiB, MiB, GiB, ...) 수정
    • M - process 를 memory 사용량으로 내림차순 정렬
    • C - process 를 CPU 사용량으로 내림차순 정렬
    • f - column 추가 및 삭제
      • ex) swap column 을 추가할 수 있다.
      • Space 키를 누르면 컬럼을 toggle 할 수 있음
      • 우측 방향키를 누르면 컬럼이 선택되어 위/아래 방향키로 위치 수정 가능
      • 좌측 방향키로 선택 해제 가능
